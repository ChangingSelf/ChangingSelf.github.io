顺便也学习了一下soft选项和hard选项的区别，假设提交历史是A到B到C，想从C撤销回到B：

  • --soft选项：HEAD 指向 B，工作区和暂存区保留 C 的修改（即 C 的更改被标记为“已暂存”），可以重新修改后提交C的修改。
  • --hard选项：HEAD 指向 B，工作区和暂存区完全恢复到 B 的状态，C 的修改被永久删除。

因此一般推荐使用前者。
